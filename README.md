# Terragrunt Aliases

Useful shortcuts for Terragrunt commands.

## Installation

Add to your shell profile:

```bash
echo "source $(pwd)/.terragrunt_aliases" >> ~/.bashrc
# or for zsh
echo "source $(pwd)/.terragrunt_aliases" >> ~/.zshrc
```

## Usage

- `tg` - terragrunt
- `tgp` - terragrunt plan
- `tga` - terragrunt apply
